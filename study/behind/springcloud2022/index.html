# Spring Cloud 入门

# Spring Cloud 介绍

Spring Cloud 是一个帮助开发者快速构建分布式系统中一些通用模式的工具，例如配置管理、服务发现、断路器、智能路由、微代理、控制总线、一次性令牌、全局锁、Leader 选举、分布式会话，集群状态等。使用 Spring Cloud 的开发人员可以快速建立起实现了这些模式的服务和应用程序，它们能够在任何分布式环境中很好地运行，包括开发者自己的笔记本电脑、裸机数据中心和云平台。

# Sprint Cloud 功能

Spring Cloud 专注于为典型用例提供良好的开箱即用体验，并提供覆盖其他用例的扩展机制。

Distributed/versioned configuration

Service registration and discovery

Routing

Service-to-service calls

Load balancing

Circuit Breakers

Global locks

Leadership election and cluster state

Distributed messaging
